SOA-01 :: Sample Project

The APIs server can be accessed by this URI: http://localhost:8080/SOA-01/soa/calculator/...

Example URI:

  • To get the current datetime of server: http://localhost:8080/SOA-01/soa/calculator/getdatetime
  • To compute the interest of 1,000,000 after a year: http://localhost:8080/SOA-01/soa/calculator/compute-interest/12/1000000
